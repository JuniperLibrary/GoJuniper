## 1. 数组（Array）是什么

数组是**固定长度、相同类型**的元素序列。数组的长度是类型的一部分。

```go
var a [3]int    // 声明一个长度为 3 的 int 数组
a[0] = 1
a[1] = 2
a[2] = 3
```

### 1.1 数组的关键特性

- **长度固定**：声明时必须指定长度，之后不能改变
- **值类型**：赋值或传参时会拷贝整个数组
- **长度是类型的一部分**：`[3]int` 和 `[5]int` 是不同的类型

```go
var a [3]int = [3]int{1, 2, 3}
var b [5]int = [5]int{1, 2, 3, 4, 5}

// 这行能编译吗？
// var c [3]int = b  // 编译错误：类型不匹配
```

### 1.2 数组 vs 切片

| 特性 | 数组 `[N]T` | 切片 `[]T` |
|------|------------|-----------|
| 长度 | 固定，声明时确定 | 动态，可增长 |
| 类型 | 长度是类型的一部分 | 长度不是类型的一部分 |
| 传参 | 值传递，拷贝整个数组 | 传引用（底层数组指针） |
| 赋值 | 拷贝整个数组 | 只拷贝描述符（指针、len、cap） |

```go
// 数组：值传递
func sumArray(a [3]int) int {
    total := 0
    for _, v := range a {
        total += v
    }
    return total
}

// 切片：引用传递
func sumSlice(s []int) int {
    total := 0
    for _, v := range s {
        total += v
    }
    return total
}
```

## 2. 数组的声明与初始化

### 2.1 基本声明

```go
var a [5]int              // 声明并初始化为零值：[0 0 0 0 0]
var b = [5]int{1, 2, 3, 4, 5}  // 声明并初始化
c := [5]int{1, 2, 3, 4, 5}     // 短变量声明
```

### 2.2 省略长度

让编译器根据初始化元素个数推断长度：

```go
a := [...]int{1, 2, 3, 4, 5}  // 长度为 5
```

### 2.3 索引初始化

只初始化特定位置的元素，其他为零值：

```go
a := [5]int{1: 10, 3: 30}  // [0 10 0 30 0]
```

### 2.4 数组字面量

```go
// 常见写法
balance := [5]float32{1000.0, 2.0, 3.4, 7.0, 50.0}

// 省略长度
balance := [...]float32{1000.0, 2.0, 3.4, 7.0, 50.0}

// 索引初始化
balance := [5]float32{1: 2.0, 3: 7.0}
```

## 3. 多维数组

Go 支持多维数组，最常见的是二维数组：

```go
// 声明二维数组
var matrix [3][2]int

// 初始化二维数组
matrix = [3][2]int{
    {1, 2},
    {3, 4},
    {5, 6},
}

// 短变量声明
matrix := [3][2]int{
    {1, 2},
    {3, 4},
    {5, 6},
}
```

### 3.1 遍历二维数组

```go
for i, row := range matrix {
    for j, val := range row {
        fmt.Printf("matrix[%d][%d] = %d\n", i, j, val)
    }
}
```

## 4. 向函数传递数组

### 4.1 值传递（会拷贝）

```go
func modifyArray(a [3]int) {
    a[0] = 100  // 修改的是副本，不影响原数组
}

func main() {
    arr := [3]int{1, 2, 3}
    modifyArray(arr)
    fmt.Println(arr)  // [1 2 3] - 原数组不变
}
```

### 4.2 传递指针（避免拷贝）

如果数组很大，拷贝开销大，可以传递指针：

```go
func modifyArrayPtr(a *[3]int) {
    a[0] = 100  // 修改的是原数组
}

func main() {
    arr := [3]int{1, 2, 3}
    modifyArrayPtr(&arr)
    fmt.Println(arr)  // [100 2 3] - 原数组被修改
}
```

### 4.3 实际开发中的选择

**一般不直接传递数组**，而是传递切片：

```go
// 推荐：传递切片
func sumSlice(s []int) int {
    total := 0
    for _, v := range s {
        total += v
    }
    return total
}

func main() {
    arr := [3]int{1, 2, 3}
    sum := sumSlice(arr[:])  // 将数组转换为切片
    fmt.Println(sum)  // 6
}
```

## 5. 指针（Pointer）是什么

指针是**存储内存地址**的变量。通过指针，你可以间接访问和修改变量的值。

```go
x := 42
p := &x    // p 是指向 x 的指针
fmt.Println(*p)  // 42 - 解引用，获取指针指向的值
*p = 100         // 通过指针修改 x
fmt.Println(x)   // 100 - x 被修改了
```

### 5.1 两个关键操作符

- `&` - 取地址：获取变量的内存地址
- `*` - 解引用：获取指针指向的值

```go
x := 42
p := &x      // p 的类型是 *int（指向 int 的指针）
fmt.Println(p)   // 0xc0000b4008（内存地址）
fmt.Println(*p)  // 42（解引用）
```

### 5.2 指针的零值

指针的零值是 `nil`，不是某个特殊的内存地址：

```go
var p *int
fmt.Println(p)   // <nil>
// fmt.Println(*p)  // 运行时错误：nil pointer dereference
```

## 6. 指针的使用场景

### 6.1 函数内修改外部变量

```go
func swap(a, b *int) {
    *a, *b = *b, *a
}

func main() {
    x, y := 1, 2
    swap(&x, &y)
    fmt.Println(x, y)  // 2 1
}
```

### 6.2 避免大结构体拷贝

```go
type LargeStruct struct {
    Data [1000]int
}

// 不推荐：拷贝整个结构体
func processBad(l LargeStruct) {
    // l 是副本，修改不影响原结构体
}

// 推荐：传递指针
func processGood(l *LargeStruct) {
    // l 指向原结构体，修改会影响原结构体
}
```

### 6.3 可选参数（用指针表示"可选"）

```go
type Config struct {
    Timeout *int    // nil 表示使用默认值
    Retries *int    // nil 表示使用默认值
}

func NewConfig(timeout, retries *int) Config {
    c := Config{}
    if timeout != nil {
        c.Timeout = timeout
    }
    if retries != nil {
        c.Retries = retries
    }
    return c
}
```

## 7. 指针与数组的关系

### 7.1 指向数组的指针

```go
arr := [3]int{1, 2, 3}
p := &arr    // p 的类型是 *[3]int

fmt.Println(*p)   // [1 2 3]
(*p)[0] = 100     // 通过指针修改数组
fmt.Println(arr)  // [100 2 3]
```

### 7.2 数组指针 vs 切片

```go
arr := [3]int{1, 2, 3}

// 指向数组的指针
pArr := &arr

// 切片（引用底层数组）
s := arr[:]

// 两者都能修改原数组，但语义不同
pArr[0] = 100  // 通过指针修改
s[1] = 200     // 通过切片修改
fmt.Println(arr)  // [100 200 3]
```

## 8. 常见陷阱

### 8.1 数组是值类型

```go
func main() {
    a := [3]int{1, 2, 3}
    b := a          // b 是 a 的副本
    b[0] = 100
    fmt.Println(a)  // [1 2 3] - a 不变
    fmt.Println(b)  // [100 2 3]
}
```

### 8.2 切片可能共享底层数组

```go
func main() {
    a := [3]int{1, 2, 3}
    b := a[:]       // b 是 a 的切片
    b[0] = 100
    fmt.Println(a)  // [100 2 3] - a 被修改了！
    fmt.Println(b)  // [100 2 3]
}
```

### 8.3 指针未初始化

```go
var p *int
// fmt.Println(*p)  // 运行时错误：nil pointer dereference

p = new(int)  // 分配内存
*p = 42
fmt.Println(*p)  // 42
```
