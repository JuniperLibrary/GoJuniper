## 1. 算术运算符

| 运算符 | 描述 | 示例 |
|--------|------|------|
| `+` | 相加 | `10 + 5` → `15` |
| `-` | 相减 | `10 - 5` → `5` |
| `*` | 相乘 | `10 * 5` → `50` |
| `/` | 相除 | `10 / 3` → `3`（整数除法） |
| `%` | 求余 | `10 % 3` → `1` |
| `++` | 自增 | `a++`（只能后置，不能前置） |
| `--` | 自减 | `a--`（只能后置，不能前置） |

### 1.1 整数除法 vs 浮点除法

```go
// 整数除法：截断小数
fmt.Println(10 / 3)    // 3
fmt.Println(-10 / 3)   // -3（向零截断）

// 浮点除法：保留小数
fmt.Println(10.0 / 3)  // 3.3333333333333335
fmt.Println(float64(10) / 3)  // 3.3333333333333335
```

### 1.2 自增自减的特殊性

Go 的 `++` 和 `--` 只能作为语句，不能作为表达式：

```go
a := 5
a++      // OK：a 变成 6
// b := a++  // 编译错误：不能用作表达式
// c := a-- + 1  // 编译错误
```

## 2. 关系运算符

| 运算符 | 描述 | 示例 |
|--------|------|------|
| `==` | 等于 | `10 == 10` → `true` |
| `!=` | 不等于 | `10 != 5` → `true` |
| `>` | 大于 | `10 > 5` → `true` |
| `<` | 小于 | `10 < 5` → `false` |
| `>=` | 大于等于 | `10 >= 10` → `true` |
| `<=` | 小于等于 | `10 <= 5` → `false` |

### 2.1 比较的限制

- 不同类型不能直接比较：`int` 和 `int64` 不能用 `==`
- 切片、map、函数不能用 `==`（只能和 `nil` 比较）
- struct 可以比较（如果所有字段都可以比较）

```go
var a int = 10
var b int64 = 10
// fmt.Println(a == b)  // 编译错误：类型不匹配
fmt.Println(a == int(b))  // true（显式转换后）

// 切片不能比较
s1 := []int{1, 2, 3}
s2 := []int{1, 2, 3}
// fmt.Println(s1 == s2)  // 编译错误
// 要比较切片，需要手动遍历或用 reflect.DeepEqual
```

## 3. 逻辑运算符

| 运算符 | 描述 | 示例 |
|--------|------|------|
| `&&` | 逻辑与（AND） | `true && false` → `false` |
| `\|\|` | 逻辑或（OR） | `true \|\| false` → `true` |
| `!` | 逻辑非（NOT） | `!true` → `false` |

### 3.1 短路求值

Go 的逻辑运算符是短路的：

```go
// &&：左边为 false，右边不执行
func sideEffect() bool {
    fmt.Println("执行了")
    return true
}

if false && sideEffect() {
    // sideEffect() 不会被调用
}

// ||：左边为 true，右边不执行
if true || sideEffect() {
    // sideEffect() 不会被调用
}
```

## 4. 位运算符

| 运算符 | 描述 | 示例 |
|--------|------|------|
| `&` | 按位与 | `0101 & 0011` → `0001` |
| `\|` | 按位或 | `0101 \| 0011` → `0111` |
| `^` | 按位异或 | `0101 ^ 0011` → `0110` |
| `&^` | 按位清除（AND NOT） | `0101 &^ 0011` → `0100` |
| `<<` | 左移 | `1 << 3` → `8` |
| `>>` | 右移 | `8 >> 2` → `2` |

### 4.1 位运算的实际用途

```go
// 权限标志
const (
    Read    = 1 << iota  // 0001 = 1
    Write               // 0010 = 2
    Exec                // 0100 = 4
)

// 检查是否有某个权限
hasRead := (permission & Read) != 0

// 添加权限
permission |= Write

// 移除权限
permission &^= Exec

// 常见：判断奇偶
if n&1 == 0 {
    fmt.Println("偶数")
} else {
    fmt.Println("奇数")
}

// 常见：乘以/除以 2 的幂
n << 3   // 等价于 n * 8
n >> 2   // 等价于 n / 4
```

## 5. 赋值运算符

| 运算符 | 描述 | 示例 |
|--------|------|------|
| `=` | 赋值 | `a = 10` |
| `+=` | 加后赋值 | `a += 5`（等价于 `a = a + 5`） |
| `-=` | 减后赋值 | `a -= 5` |
| `*=` | 乘后赋值 | `a *= 5` |
| `/=` | 除后赋值 | `a /= 5` |
| `%=` | 取余后赋值 | `a %= 5` |
| `&=` | 按位与后赋值 | `a &= 5` |
| `\|=` | 按位或后赋值 | `a \|= 5` |
| `^=` | 按位异或后赋值 | `a ^= 5` |
| `<<=` | 左移后赋值 | `a <<= 2` |
| `>>=` | 右移后赋值 | `a >>= 2` |

## 6. 运算符优先级

优先级从高到低：

| 优先级 | 运算符 |
|--------|--------|
| 5 | `* / % << >> & &^` |
| 4 | `+ - \| ^` |
| 3 | `== != < <= > >=` |
| 2 | `&&` |
| 1 | `\|\|` |

**建议**：不确定优先级时，用括号明确：

```go
// 不推荐
result := a + b * c

// 推荐（清晰）
result := a + (b * c)
```

## 7. if 条件语句

### 7.1 基本语法

```go
if condition {
    // 条件为 true 时执行
} else {
    // 条件为 false 时执行
}
```

### 7.2 if-else if-else

```go
if score >= 90 {
    fmt.Println("优秀")
} else if score >= 80 {
    fmt.Println("良好")
} else if score >= 60 {
    fmt.Println("及格")
} else {
    fmt.Println("不及格")
}
```

### 7.3 if 带初始化语句

Go 的 if 可以带一个初始化语句：

```go
if err := doSomething(); err != nil {
    fmt.Println("出错了:", err)
}

// 等价于
err := doSomething()
if err != nil {
    fmt.Println("出错了:", err)
}
```

**常见用途**：

```go
// 读取文件
if f, err := os.Open("file.txt"); err != nil {
    fmt.Println("打开文件失败:", err)
} else {
    defer f.Close()
    // 使用文件
}

// 类型断言
if s, ok := i.(string); ok {
    fmt.Println("字符串:", s)
}
```

### 7.4 条件必须是布尔类型

```go
x := 5
// if x {  // 编译错误：x 不是 bool 类型
//     fmt.Println("非零")
// }

if x != 0 {
    fmt.Println("非零")
}
```

## 8. switch 多分支

### 8.1 基本语法

```go
switch expression {
case value1:
    // 匹配 value1
case value2:
    // 匹配 value2
default:
    // 都不匹配
}
```

### 8.2 无条件 switch（代替 if-else if）

```go
switch {
case score >= 90:
    fmt.Println("优秀")
case score >= 80:
    fmt.Println("良好")
case score >= 60:
    fmt.Println("及格")
default:
    fmt.Println("不及格")
}
```

### 8.3 多值匹配

```go
switch day {
case "Saturday", "Sunday":
    fmt.Println("周末")
case "Monday", "Tuesday", "Wednesday", "Thursday", "Friday":
    fmt.Println("工作日")
default:
    fmt.Println("无效")
}
```

### 8.4 类型 switch

```go
func describe(i interface{}) {
    switch v := i.(type) {
    case int:
        fmt.Printf("整数: %d\n", v)
    case string:
        fmt.Printf("字符串: %q\n", v)
    case bool:
        fmt.Printf("布尔: %v\n", v)
    default:
        fmt.Printf("未知类型: %T\n", v)
    }
}
```

### 8.5 fallthrough（穿透）

Go 的 switch 默认不会穿透到下一个 case（和 C/Java 不同）。如果需要穿透，用 `fallthrough`：

```go
switch x {
case 1:
    fmt.Println("一")
    fallthrough  // 穿透到下一个 case
case 2:
    fmt.Println("二")  // 如果 x=1，这里也会执行
case 3:
    fmt.Println("三")
}
```

**注意**：`fallthrough` 只能穿透到下一个 case，不能穿透多个。

### 8.6 switch 中的初始化语句

```go
switch os := runtime.GOOS; os {
case "linux":
    fmt.Println("Linux 系统")
case "darwin":
    fmt.Println("macOS 系统")
case "windows":
    fmt.Println("Windows 系统")
default:
    fmt.Println("其他系统:", os)
}
```

## 9. for 循环

Go 只有 `for` 一种循环，但可以实现 while、do-while 的效果。

### 9.1 基本 for 循环

```go
for i := 0; i < 10; i++ {
    fmt.Println(i)
}
```

### 9.2 while 风格

```go
// 等价于 while condition { ... }
n := 1
for n < 1000 {
    n *= 2
}
```

### 9.3 无限循环

```go
// 死循环（用 break 退出）
for {
    // 处理任务
    if done {
        break
    }
}
```

### 9.4 for 循环的变体

```go
// 省略 init 和 post
for condition {
    // 等价于 while condition
}

// 省略所有部分
for {
    // 无限循环
}
```

## 10. range 迭代

`range` 用于遍历切片、数组、map、字符串、channel。

### 10.1 遍历切片/数组

```go
nums := []int{10, 20, 30}

// 只需要值
for _, v := range nums {
    fmt.Println(v)  // 10, 20, 30
}

// 需要索引和值
for i, v := range nums {
    fmt.Printf("索引 %d: 值 %d\n", i, v)
}

// 只需要索引
for i := range nums {
    fmt.Println(i)  // 0, 1, 2
}
```

### 10.2 遍历字符串

```go
// 字符串按 rune 遍历
for i, r := range "Hello, 你好" {
    fmt.Printf("索引 %d: %c (Unicode: %U)\n", i, r, r)
}
// 输出：
// 索引 0: H (Unicode: U+0048)
// 索引 1: e (Unicode: U+0065)
// 索引 2: l (Unicode: U+006C)
// 索引 3: l (Unicode: U+006C)
// 索引 4: o (Unicode: U+006F)
// 索引 5: , (Unicode: U+002C)
// 索引 6:   (Unicode: U+0020)
// 索引 7: 你 (Unicode: U+4F60)
// 索引 10: 好 (Unicode: U+597D)
```

### 10.3 遍历 map

```go
m := map[string]int{"a": 1, "b": 2, "c": 3}

for k, v := range m {
    fmt.Printf("%s = %d\n", k, v)
}

// 只需要 key
for k := range m {
    fmt.Println(k)
}

// 只需要 value
for _, v := range m {
    fmt.Println(v)
}
```

### 10.4 range 的陷阱

```go
// 陷阱：修改切片元素
nums := []int{1, 2, 3}
for _, v := range nums {
    v *= 2  // 修改的是副本，原切片不变
}
fmt.Println(nums)  // [1 2 3]

// 正确：通过索引修改
for i := range nums {
    nums[i] *= 2
}
fmt.Println(nums)  // [2 4 6]
```

## 11. break 和 continue

### 11.1 break

```go
// 跳出当前循环
for i := 0; i < 10; i++ {
    if i == 5 {
        break  // 跳出循环
    }
    fmt.Println(i)  // 0, 1, 2, 3, 4
}
```

### 11.2 continue

```go
// 跳过当前迭代
for i := 0; i < 10; i++ {
    if i%2 == 0 {
        continue  // 跳过偶数
    }
    fmt.Println(i)  // 1, 3, 5, 7, 9
}
```

### 11.3 带标签的 break/continue

```go
// 多层循环中使用标签
outer:
    for i := 0; i < 10; i++ {
        for j := 0; j < 10; j++ {
            if i*j > 50 {
                break outer  // 跳出外层循环
            }
            fmt.Printf("%d*%d=%d\n", i, j, i*j)
        }
    }
```
